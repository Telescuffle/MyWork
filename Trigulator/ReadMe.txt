How to use Trigulator: The C++ trigonometry calculator

Open the Trigulator application file.

Once the program is open you will be on the main menu. 

Just select using the assigned number for the menu option that most closely meets your given problem. 

The system will then ask you to enter your values. For any values you do not have, please enter a 0.

Once your calculation is complete, they system will offer you to return to the main menu if you have 
another calculation you want to make. 

Once again, please use to corresponding number to the choice you want to make.
